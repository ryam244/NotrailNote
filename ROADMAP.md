# NotrailNote ロードマップ

## 概要

NotrailNoteは、AI搭載のMarkdownエディタで、GitHubと連携してノートを安全に管理できるアプリです。

---

## 優先度マトリクス

| 優先度 | 機能 | 理由 |
|--------|------|------|
| 🔴 高 | バグ修正・安定性 | ユーザー維持に必須 |
| 🔴 高 | GitHub Pull機能 | 双方向同期の完成 |
| 🟡 中 | フォルダ機能 | 整理機能の要望多 |

## リスク・課題

| リスク | 対策 |
|--------|------|
| GitHub API制限 | レート制限対応、キャッシュ活用 |
| AI APIコスト | 使用量制限、効率的なプロンプト |
| 競合サービス | 独自機能（GitHub連携）で差別化 |
| データ消失 | 自動バックアップ、同期確認UI |
